WHO AM I?

I'm a IB 12 student who hopefully can go into forensic sciences. At the moment, I enjoy making concept art + character design, doing hobby coding, and watching movies.

I've done a lot of design work for online closed species communities, with running my own ARPG Auteryx. I also created various HTML + CSS resources for closed species communities, and worked on various web projects.

WHAT'S THIS SITE FOR?

This is my website that I'm coding. I hope to use it as both a personal expression of myself on the internet, and also keep it sleek and professional. It will contain things like:

  • Links to all projects
  • Current CV
  • Art portfolio (?)
  • Blog
  • Sneak peaks for future projects/ideas
Layout + stuff is done by me, but blog function is supported by a modified version of the charadex framework.
